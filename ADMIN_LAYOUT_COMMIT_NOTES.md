# Admin/Layout Commit Notes

This commit is the non-SEO follow-up slice for admin, layout, review, metadata, and sitemap-documentation cleanup.

Included:

- Admin login presentation polish and dashboard font/radius guardrails.
- Site metadata image updates and Inter font support.
- Reviews page, review aggregate data, review sitemap entry, and homepage trust/review presentation.
- Static page structured-data refinements for about, contact, blog, and gallery surfaces.
- Redirect and sitemap documentation updates that are not tied to chatbot, members, or shopping logic.

- Chatbot prompts, chatbot UI, chatbot APIs, chatbot scripts, chatbot database tables, and chatbot storage.
- Members, subscribers, unsubscribe, email campaign, and email delivery logic.
- Shopping, product catalog, cart, order, PayFast, and product admin logic.
- Package dependency/script changes that belong to the excluded chatbot, email, or shopping work.

Those excluded files may remain in the working tree, but they are intentionally not part of this commit.
